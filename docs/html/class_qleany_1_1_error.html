enum Qleany::Error::Status

The Status enum defines the possible error statuses.

Enumerator
Ok 

The operation succeeded without any issues.

Warning 

The operation succeeded but with some non-fatal issues or warnings.

The operation failed with a critical error.

The operation failed with a fatal error.
